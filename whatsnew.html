NGFB version 7.6 provides a new 'Licenses' settings page to allow for future plugin extensions. Although most of the setting pages and features may not appear to have changed from v7.5.5, the underlying code - to support future plugin extensions - has undergone important revisions. The 'Network' multisite sub-menu has also been renamed 'Advanced', and provides a few more cache related options for Network sites / blogs.

Version 7.6.1 fixes a static class reference in 'NgfbSharing' for PHP v5.2 and older.

Version 7.6.2 fixed a condition that would prevent sharing button JavaScripts from being loaded on index pages without posts.

Version 7.6.3 includes small updates to the settings help texts.

Version 7.6.3.1 includes a fix for empty / missing Polylang post language, and adds support for the new pll_get_post_language() function (Pro version).

Version 7.6.3.2 fixes user ID detection on author index webpages, and improves the "Version Information" side metabox (now with information on extension plugins as well).

Version 7.6.3.3 updates the minimum short URL value from 21 to 22, and allows for longer Tweet text when using Twitter's shortening service.
